# Terminologie

Om de terminologie het makkelijkste uit te leggen vergelijken we onze database met een Excelfile.

* Excelfile = Database
* Sheets van de Excelfile = Tables
* Kolommen van de Excelfile = Columns
* Rijen van de Excelfile = Rows
